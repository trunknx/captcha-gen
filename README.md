# Foobar

Captcha-gen is a tool to generate captcha. Build on opencv library

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```bash
pip install -r requirements.txt
```

## Usage

```python
python main.py
```
